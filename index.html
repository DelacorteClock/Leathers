Programming Projects By TheLeathers

LeatherPants

React Native application enabling anonymous conversation about fashion. Simplistically modern interface for fashion lovers.

EdgyEvents

Progressive web app with React for displaying Google Calendar events. Integrates Google oAuth for security and Recharts for statistical visualisation.

ListMake

System made with help of jQuery which allows users to make and modify simple lists in the browser by adding, deleting, marking and re-ordering items.

Station Info

JS based application designed to display information about the lines and stations of a famous railway. Chimes indicate clicks and the artistic theme is inspired by the windows of trains.

Standard PVC

Website made with plain HTML/CSS for a fictional PVC manufacturer called 'Standard PVC'. Features include a contact section and a projects page.

Angular NuMuseum

A variation of the React-based NuMuseum with a red theme and Angular framework. Horizontal gallery and profile section are key features.

NuMuseum

Full stack application allowing users to access information about items in TheLeathers' collection of art and design. It is comprised of two key parts: the NuMuseum API and the NuMuseum Front End. I wanted to create a personal project linking my previous art/design work to my future programming career.
